================================================================================
SOLUCIÓN DE PROBLEMAS
================================================================================

ERROR: "No module named 'pyodbc'"
SOLUCIÓN:
  pip install pyodbc
  # En Windows, podría necesitar C++ Build Tools
  # En Linux: sudo apt-get install python3-dev unixodbc-dev
  # En macOS: brew install unixodbc

ERROR: "Unable to connect to SybaseIQ"
SOLUCIÓN:
  1. Verifica que el cliente ODBC está instalado
  2. Prueba la conexión con isql:
     isql -S tu-servidor -U usuario -P contraseña
  3. Verifica el servidor y puerto
  4. Comprueba permisos de usuario

ERROR: "ANTHROPIC_API_KEY not found"
SOLUCIÓN:
  export ANTHROPIC_API_KEY="sk-ant-..."
  # O crea archivo .env con la clave

ERROR: "Certificate verify failed"
SOLUCIÓN:
  # En Windows, podría ser problema de certificados SSL
  pip install certifi
  # O desactiva verificación (NO RECOMENDADO en producción):
  import ssl
  ssl._create_default_https_context = ssl._create_unverified_context

ERROR: "Connection timeout"
SOLUCIÓN:
  1. Aumenta timeout en config:
     timeout=60  # aumentar de 30 a 60
  2. Verifica conectividad de red:
     ping tu-servidor.com
  3. Comprueba firewall
  4. Verifica puerto (por defecto 5000)
